## 题目描述

题目链接:[15. 三数之和 - 力扣（LeetCode） (leetcode-cn.com)](https://leetcode-cn.com/problems/3sum/)

给你一个包含 `n` 个整数的数组` nums`，判断 `nums` 中是否存在三个元素 $$a$$，$$b$$，$$c$$ ，使得 $$a + b + c = 0 ?$$请你找出所有和为 `0` 且不重复的三元组。

**注意**：答案中不可以包含重复的三元组。

示例1：

```
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
```

示例2：

```
输入：nums = []
输出：[]
```

示例3：

```
输入：nums = [0]
输出：[]
```



**提示**:

- $0 \le nums.length \le 3000$
- $-10^5 \le nums[i] \le 10^5$
