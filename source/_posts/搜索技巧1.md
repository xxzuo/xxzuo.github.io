# 利用数组确定搜索方向



我们可以新建方向数组，这样 dfs 里不用自己手写搜索方向

```java
int[] dx = new int[]{1, 0, -1, 0};
int[] dy = new int[]{0, 1, 0, -1};

for(int n = 0; n< 4; n++){
    int nx = i + dx[n];
    int ny = j + dy[n];
}

```
